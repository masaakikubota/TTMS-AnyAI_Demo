# UCT TTMS Dashboard Mock

Looker Studioを完全に再現したHTMLモックアップ。GitHub Pagesでホスティングし、リンクで共有可能。

## 技術スタック
- **HTML/CSS/JS**（ビルド不要、静的ファイルのみ）
- **Chart.js**（CDN） — チャート描画
- **Google Fonts**（CDN） — Looker Studioのタイポグラフィ再現
- **デプロイ**: GitHub Pages（mainブランチ / ルートから配信）
- **ローカル確認**: `python3 -m http.server 8000` で `http://localhost:8000` から確認

## ページ構成（8ページ = 2カテゴリ × 4ページ）

カテゴリ（BC / FC）ごとに4ページを持つ。ページロジックは共通で、カテゴリをパラメータとして受け取り描画する。

### ページ定義
| # | ページ | 描画ロジック | 内容 | 対応レイヤー |
|---|-------|------------|------|------------|
| P1 | Overview | js/pages/overview.js | GMV結果サマリ（TTS/Shopee）、SKU・Video・Live別パフォーマンス | Layer 1: 結果 |
| P2 | Touchpoint | js/pages/touchpoint.js | タッチポイント遷移、ACCファネル、検索行動、市場ポジション | Layer 2: 行動兆候 |
| P3 | Content | js/pages/content.js | 訴求軸構成比、コメント言及分析、Sentiment | Layer 3: 質 |
| P4 | Correlation | js/pages/correlation.js | レイヤー間の相関分析（A1〜D3） | 横断分析 |

各ページは `render(category, container)` 関数をexportし、`app.js` が現在のカテゴリ（`'bc'` / `'fc'`）を渡して呼び出す。

## Looker Studio UI再現の方針

### デザイントークン（Unicharmカラー適用）
| 要素 | 値 |
|-----|-----|
| **暗いアクセント** | `#3C459A`（Unicharm紺）— スコアカードヘッダー、テーブルヘッダー、サイドバーアクティブ |
| **明るいアクセント** | `#F18B00`（Unicharmオレンジ）— アクティブタブ下線、ハイライト、増加指標 |
| 背景色 | `#f0f0f0`（ページ）/ `#ffffff`（メインカード） |
| フォント | `Google Sans`, `Roboto`, sans-serif |
| メインカード角丸 | `12px` |
| メインカード影 | `0 1px 3px rgba(0,0,0,0.12), 0 1px 2px rgba(0,0,0,0.08)` |
| スコアカード角丸 | `8px` |
| スコアカードボーダー | `1px solid #e0e0e0` |
| ページタブ | 上部水平（アクティブ=オレンジ下線 `#F18B00`） |
| サイドバー | 左フロート（アイコン+ラベル、アクティブ=オレンジテキスト `#F18B00`） |

### スコアカードスタイル（スクリーンショット準拠）
```
┌───────────────────────┐
│    Label              │ ← 暗い背景 (#3C459A)、白テキスト、小さめフォント
├───────────────────────┤
│                       │
│   1,234,567           │ ← 白背景、大きな数値、中央揃え
│                       │
└───────────────────────┘
```

### テーブルスタイル（スクリーンショット準拠）
- タイトル: テーブル上部に中央揃え、太字
- ヘッダー行: 白背景、太字テキスト、下ボーダー
- 数値セル: 一部にダーク背景（`#3C459A`）+白テキストで強調表示
- 右上にフィルター・メニューアイコン

### チャートスタイル
- 折れ線/棒: カスタムパレット (`#3C459A`, `#F18B00`, `#4CAF50`, `#E91E63`, `#00BCD4`, `#9C27B0`)
- スコアカード: 大きな数値 + 前週比の矢印（`#4CAF50`↑ / `#E91E63`↓）

## フィルター仕様
上部ページタブの中でカテゴリ（BC/FC）も切り替え。フィルターバーにはそれ以外の条件を配置。

| フィルター | 種類 | 選択肢 | アイコン |
|-----------|------|--------|---------|
| Category | ドロップダウン | BC / FC | 📦 |
| Period | 日付範囲 | 週次単位（デフォルト: 直近4週） | 📅 |
| Benchmark | ドロップダウン | Top 5 / Top 25 | 📊 |

## モックデータの方針
- `mock-data.js` に全データを定義（外部API呼び出しなし）
- **BC / FC** それぞれのデータセットを持つ
- 2025/12〜2026/01の週次データ（8週分程度）をハードコード
- 提案書のサンプルデータ（reference/samples/）を参考に現実的な数値を使用
- ベンチマークデータはBrand値の±20%程度で生成
